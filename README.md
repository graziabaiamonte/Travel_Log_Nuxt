# Travel Log - Nuxt

Full Stack app that allows users to keep track of the places they have visited and view them on a map.

## Tech

- Nuxt
- Vue
- Pinia
- Drizzle ORM
- MapLibre GL
- Zod
- Tailwind CSS
- DaisyUI
- Husky
- Docker
- GitHub Action (CI/CD)
